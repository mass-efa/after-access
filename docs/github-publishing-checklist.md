## Link Check

Confirm that these links resolve after publishing:

- `README.md` prompt links
- `docs/safety-and-compliance-guide.md`
- `docs/github-publishing-checklist.md`
- Each prompt file's repository link

If you publish under a different owner or repo name, replace this URL everywhere:

`https://github.com/mass-efa/after-access`

## Content Check

- Keep the confidentiality and IP cautions intact.
- Keep the "not legal advice" language intact.
- Confirm no actual employer-specific confidential content is included.
- Confirm no personal data, credentials, source code, internal emails, or proprietary docs are included.
- Preview the README on GitHub before sharing.
